# portfolio_manager - 组合管理技能

基于现代投资组合理论的组合分析与管理系统，帮助投资者优化资产配置、控制风险敞口、实现动态再平衡。

> **skill名称**: `portfolio_manager` (组合管理)
> **理论基础**: 现代投资组合理论(MPT)、风险平价、杠铃策略
> **适用场景**: 组合监控、再平衡决策、风险预算分配

---

## 概述

本技能是一个完整的组合管理系统，核心特点：

- **组合分析**: 收益率、波动率、最大回撤、夏普比率
- **相关性监控**: 识别高度相关持仓，避免风险集中
- **风险敞口**: 行业/风格/市场多维度的风险分解
- **再平衡建议**: 基于偏离度的动态调整建议
- **杠铃策略**: 安全端/风险端的杠铃配置

### 核心指标

| 指标 | 说明 | 健康范围 |
|------|------|---------|
| 年化收益率 | 组合年度收益 | >10%优秀 |
| 年化波动率 | 收益波动程度 | <20%适中 |
| 最大回撤 | 峰值到谷底跌幅 | <15%可控 |
| 夏普比率 | 风险调整后收益 | >0.5良好 |
| 贝塔系数 | 相对市场波动 | 0.8-1.2平衡 |
| 行业集中度 | 单一行业占比 | <30%分散 |

## 使用方法

### 命令行运行

```bash
cd /Users/lijian/.agents/skills/portfolio_manager
python run.py --portfolio portfolio.json
```

### 组合文件格式

```json
{
  "holdings": [
    {"ts_code": "000001.SZ", "name": "平安银行", "quantity": 1000, "cost": 10.5},
    {"ts_code": "600519.SH", "name": "贵州茅台", "quantity": 100, "cost": 1680.0}
  ],
  "cash": 50000,
  "total_capital": 1000000
}
```

### Python调用

```python
from portfolio_manager.portfolio import PortfolioManager

# 初始化
manager = PortfolioManager(pro_api)

# 加载持仓
portfolio = manager.load_portfolio('portfolio.json')

# 组合分析
analysis = manager.analyze_portfolio(portfolio)

# 再平衡建议
rebalance = manager.get_rebalance_suggestion(portfolio)
```

## 功能模块

### 1. 组合分析 (PortfolioAnalyzer)

计算组合的各项指标：
- 收益率分析（总收益/年化/滚动）
- 风险指标（波动率/回撤/VaR）
- 风险调整后收益（夏普/索提诺/卡玛）
- 相对基准表现（Alpha/Beta/信息比率）

### 2. 相关性监控 (CorrelationMonitor)

- 持仓间相关系数矩阵
- 识别高度相关组合（>0.7警告，>0.8严重）
- 行业集中度分析
- 风格敞口分析（价值/成长/大盘/小盘）

### 3. 再平衡引擎 (RebalanceEngine)

- 目标权重设定
- 偏离度计算
- 再平衡触发判断（偏离>5%或定期）
- 交易费用估算

### 4. 风险预算 (RiskBudget)

- 风险平价计算
- 风险贡献分解
- 风险预算分配建议
