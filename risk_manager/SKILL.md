# risk_manager - 风险管理

基于多维度风控指标的风险管理系统，提供止损追踪、回撤监控、风险预算等功能。

> **skill名称**: `risk_manager` (风险管理)
> **风控层次**: 技术止损/时间止损/波动率止损/组合止损
> **适用场景**: 日常风险监控、止损决策、仓位优化

---

## 概述

本技能是一个完整的风险管理系统，核心特点：

- **止损追踪**: 四层止损体系（技术/时间/波动率/组合）
- **回撤监控**: 实时和历史最大回撤
- **风险预算**: VaR/ES计算、风险平价
- **仓位优化**: 凯利公式最优仓位

### 四层止损体系

| 层次 | 触发条件 | 适用场景 |
|------|---------|---------|
| 技术止损 | 趋势股-8%，价值股-20% | 右侧交易 |
| 时间止损 | 4-8周未达预期 | 左侧交易 |
| 波动率止损 | ATR动态调整 | 高波动标的 |
| 组合止损 | 回撤10%减仓/20%清仓 | 组合层面 |

## 使用方法

### 命令行运行

```bash
cd /path/to/skills/risk_manager
python run.py
```

### Python调用

```python
import sys
sys.path.insert(0, '/path/to/skills')

from risk_manager.risk_manager import RiskManager
import tushare as ts

pro = ts.pro_api()
rm = RiskManager(pro)

# 持仓风险扫描
holdings = [
    {'ts_code': '000001.SZ', 'quantity': 1000, 'cost': 10.5},
    {'ts_code': '600519.SH', 'quantity': 100, 'cost': 1680.0}
]
result = rm.scan_portfolio_risk(holdings)
rm.print_risk_report(result)

# 凯利公式计算
kelly = rm.kelly_criterion(win_rate=0.55, avg_win=0.15, avg_loss=0.08)
print(f"最优仓位: {kelly*100:.1f}%")
```

## 核心功能

### 1. 止损检查

- **严格止损**: -5%
- **常规止损**: -8%
- **宽松止损**: -12%

### 2. VaR计算

```python
var_95 = rm.calculate_var(returns, 0.95)
var_99 = rm.calculate_var(returns, 0.99)
```

### 3. 凯利公式

```
f* = (bp - q) / b

其中:
b = 平均盈利 / 平均亏损 (赔率)
p = 胜率
q = 1 - p (败率)
```
