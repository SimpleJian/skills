# futures_trend - 商品期货趋势跟踪选股策略

针对100万资金规模的进取型趋势跟踪策略，采用"三层筛选+综合评分"框架筛选商品期货标的。

> **skill名称**: `futures_trend` (商品期货趋势跟踪)
> **策略类型**: 趋势跟踪 / CTA策略
> **适用市场**: 国内四大期货交易所（上期所、大商所、郑商所、广期所）
> **资金规模**: 100万
> **风险偏好**: 进取型

---

## 概述

本策略是国内商品期货市场的趋势跟踪选股系统，核心特点：

- **三层筛选框架**：流动性 → 趋势方向 → 趋势强度
- **多维度量化评分**：动量30% + 趋势强度30% + 量价配合25% + 波动风险15%
- **动态多空配置**：根据市场环境调整净敞口
- **组合风险控制**：板块分散 + 相关性控制

### 适用场景

| 市场环境 | 策略表现 | 净敞口建议 |
|---------|---------|-----------|
| 强牛市 | 优秀 | +100% 多头 |
| 弱牛市 | 良好 | +60% 净多头 |
| 震荡市 | 较差 | 降低仓位或观望 |
| 弱熊市 | 良好 | -60% 净空头 |
| 强熊市 | 优秀 | -100% 空头 |

## 快速开始

### 1. 标准选股（震荡市）

```bash
cd /path/to/skills/futures_trend

# 标准运行
python3 run.py

# 指定市场环境
python3 run.py --env 强牛

# 指定资金规模
python3 run.py --capital 200

# 调整ADX阈值
python3 run.py --adx 30
```

### 2. 分析指定合约

```bash
python3 run.py --code CU.SHF    # 分析沪铜
python3 run.py --code RB.SHF    # 分析螺纹钢
python3 run.py --code M.DCE     # 分析豆粕
```

### 3. 代码中使用

```python
import tushare as ts
from futures_selector import FuturesSelector

pro = ts.pro_api()
selector = FuturesSelector(pro, total_capital=1000000)

# 执行选股
result = selector.select_contracts(
    market_env='震荡',  # 市场环境
    adx_threshold=25    # ADX阈值
)

# 查看结果
print(result['core_long'])     # 核心多头
print(result['core_short'])    # 核心空头
```

## 模块说明

### liquidity_filter.py - 流动性筛选模块

**核心类**: `LiquidityFilter`

**主要方法**:

| 方法 | 说明 |
|------|------|
| `filter_by_liquidity(top_n)` | 第一层流动性筛选 |
| `calculate_liquidity_metrics(ts_code)` | 计算流动性指标 |
| `get_commodity_category(ts_code)` | 获取商品类别 |

**筛选标准**:
- 日均成交量 ≥ 10万手
- 日均持仓量 ≥ 5万手
- 量仓比 1-2 为健康状态

### trend_direction.py - 趋势方向判定模块

**核心类**: `TrendDirection`

**主要方法**:

| 方法 | 说明 |
|------|------|
| `filter_by_trend(df_liquidity, adx_threshold)` | 第二层趋势方向筛选 |
| `determine_trend_direction(df)` | 判定趋势方向 |
| `calculate_adx(df)` | 计算ADX指标 |

**判定标准**:

| 趋势状态 | 价格与均线 | ADX | 策略立场 |
|---------|-----------|-----|---------|
| 明确多头 | 价格>MA20>MA60 | >25 | 纳入多头池 |
| 明确空头 | 价格<MA20<MA60 | >25 | 纳入空头池 |
| 震荡 | 均线缠绕 | <20 | 剔除 |

### trend_strength.py - 趋势强度量化模块

**核心类**: `TrendStrength`

**主要方法**:

| 方法 | 说明 |
|------|------|
| `calculate_comprehensive_score(ts_code, direction)` | 计算综合评分 |
| `calculate_momentum(df)` | 计算动量指标 |
| `calculate_cmf(df)` | 计算CMF资金流向 |
| `calculate_atr(df)` | 计算ATR波动率 |

**权重配置**:

| 指标类别 | 权重 | 细分指标 |
|---------|------|----------|
| 动量类 | 30% | 长周期动量(20%) + 短周期动量(10%) |
| 趋势强度 | 30% | ADX(20%) + 布林带斜率(10%) |
| 量价配合 | 25% | CMF(15%) + PVT(10%) |
| 波动风险 | 15% | ATR(10%) + 最大回撤(5%) |

### portfolio_builder.py - 组合构建模块

**核心类**: `PortfolioBuilder`

**主要方法**:

| 方法 | 说明 |
|------|------|
| `build_portfolio(df_long, df_short, market_env)` | 构建投资组合 |
| `get_commodity_sector(ts_code)` | 获取商品板块 |
| `check_correlation(code1, code2)` | 检查品种相关性 |

**约束条件**:
- 同板块最多2个品种
- 产业链高度相关品种最多1个
- 单品种仓位上限20万

### futures_selector.py - 主选股模块

**核心类**: `FuturesSelector`

**主要方法**:

| 方法 | 说明 |
|------|------|
| `select_contracts(market_env, adx_threshold)` | 执行完整选股流程 |

## 多空配置策略

### 市场环境判定与配置

| 市场环境 | 判定标准 | 净敞口 | 配置方案 |
|---------|---------|--------|---------|
| 强牛市 | 商品指数>MA20>MA60, ADX>30 | +100% | 100%多头 |
| 弱牛市 | 商品指数>MA20>MA60, ADX 20-30 | +60% | 多头为主，少量对冲 |
| 震荡市 | 均线缠绕, ADX<20 | 0% | 多空平衡或观望 |
| 弱熊市 | 商品指数<MA20<MA60, ADX 20-30 | -60% | 空头为主，少量对冲 |
| 强熊市 | 商品指数<MA20<MA60, ADX>30 | -100% | 100%空头 |

### 板块优先级

**高优先级板块**：
- 有色金属（铜、铝、镍）：新能源需求、供应约束
- 能源化工（PTA、甲醇、原油）：地缘风险、产能整合
- 农产品（豆粕、玉米）：气候变化、粮食安全

**中优先级板块**：
- 黑色系（螺纹钢、铁矿石）：房地产调整、基建支撑
- 贵金属（黄金、白银）：美联储政策、避险需求

## 风险管理

### 止损设置

| 止损类型 | 触发条件 | 执行方式 |
|---------|---------|---------|
| ATR止损 | 价格反向触及2-3倍ATR | 动态调整，趋势初期较紧 |
| 均线止损 | 收盘价跌破MA20/MA60 | 趋势可能转折，果断离场 |
| 趋势反转止损 | ADX跌破25或DI交叉 | 趋势强度丧失，离场观望 |

### 仓位管理

- **首次建仓**：信号触发时30%目标仓位
- **加仓条件**：趋势确认且浮盈>5%，加仓至60%
- **满仓条件**：趋势强劲且风险可控，可满仓
- **单品种上限**：不超过总资金20%（20万）

## 注意事项

### 频率限制说明 ⚠️

Tushare API 有频率限制（免费用户通常每分钟 500 次）。策略已内置以下机制：

1. **自动延时控制**：API 调用间隔自动控制在 0.15 秒以上
2. **自动重试**：遇到"每分钟最多访问"错误时，自动等待 10 秒后重试
3. **分批获取**：合约数据分批获取，每批之间自动延时
4. **缓存优化**：基础数据（合约列表、交易日历）缓存复用

**运行时间参考**：
- 全市场 70+ 个品种扫描：约 2-3 分钟
- 三层筛选+评分：约 3-5 分钟
- **总计：5-8 分钟**

**建议**：
- 首次运行或数据过期时，可能需要较长时间，请耐心等待
- 若遇到 "等待 X 秒..." 提示，属正常现象
- 可避开交易时段（9:00-15:00，21:00-次日2:30）运行选股

### 策略风险

1. **数据限制**：Tushare期货数据覆盖范围有限，部分品种数据可能不完整

2. **交易时间**：期货市场夜盘交易需特别关注，策略信号基于日线

3. **保证金管理**：期货杠杆交易，需预留充足保证金应对波动

4. **移仓换月**：主力合约切换时需及时处理，避免流动性风险

5. **策略风险**：
   - 趋势跟踪策略在震荡市表现较差
   - 进取型策略波动较大，需有心理准备
   - 历史回测不代表未来表现
