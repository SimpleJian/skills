# macro_analysis - 宏观分析技能

基于Tushare宏观经济数据的美林投资时钟分析系统，帮助投资者识别经济周期、判断市场趋势、优化大类资产配置。

> **skill名称**: `macro_analysis` (宏观分析)
> **分析框架**: 美林投资时钟
> **数据频率**: 月度/季度更新
> **适用场景**: 战略资产配置、市场趋势判断

---

## 概述

本技能是一个完整的宏观经济分析系统，核心特点：

- **美林投资时钟**: 基于GDP和通胀的经济周期四阶段判断
- **多维度指标**: GDP/CPI/PPI/PMI/利率/社融/M2等关键宏观指标
- **趋势分析**: 指标同比/环比变化趋势
- **资产配置建议**: 根据周期阶段给出大类资产权重建议

### 经济周期四阶段

| 阶段 | GDP | 通胀(CPI) | 最优资产 | 当前A股风格 |
|------|-----|-----------|---------|------------|
| 复苏期 | ↑ | ↓ | 股票 | 成长/周期 |
| 过热期 | ↑ | ↑ | 商品 | 周期/金融 |
| 滞胀期 | ↓ | ↑ | 现金 | 防御/消费 |
| 衰退期 | ↓ | ↓ | 债券 | 成长/防御 |

---

## 核心功能

### 1. 经济周期判断

基于GDP增速和CPI变化趋势，判断当前经济所处阶段：

```python
phase, confidence = analyzer.detect_economic_phase()
# 输出: ("复苏期", 0.75)
```

### 2. 宏观指标监控

跟踪关键宏观指标：

| 指标 | 频率 | 数据来源 | 判断标准 |
|------|------|---------|---------|
| GDP | 季度 | cn_gdp | >6%强劲，4-6%平稳，<4%疲软 |
| CPI | 月度 | cn_cpi | >3%高通胀，1-3%温和，<1%通缩 |
| PPI | 月度 | cn_ppi | 转正企业盈利改善 |
| PMI | 月度 | cn_pmi | >50扩张，<50收缩 |
| M2 | 月度 | cn_m | 增速反映流动性 |
| 社融 | 月度 | sf_month | 增速反映信用扩张 |
| LPR | 月度 | shibor_lpr | 利率方向 |

### 3. 资产配置建议

根据经济周期给出建议配置：

```
复苏期配置建议:
  股票: 60-80% (超配)
  债券: 20-30% (标配)
  现金: 5-10% (低配)
  商品: 5-10% (低配)
```

## 使用方法

### 命令行运行

```bash
cd /path/to/skills/macro_analysis
python run.py
```

### Python调用

```python
import sys
sys.path.insert(0, '/path/to/skills')
from macro_analysis.macro_analyzer import MacroAnalyzer
import tushare as ts

# 初始化
pro = ts.pro_api()
analyzer = MacroAnalyzer(pro)

# 完整分析
result = analyzer.full_analysis()

# 获取周期判断
phase, confidence = analyzer.detect_economic_phase()

# 获取资产配置建议
allocation = analyzer.get_asset_allocation()
```
